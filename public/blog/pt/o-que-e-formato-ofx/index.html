O que é o Formato OFX? Um Guia Completo

Publicado em março de 2026 · 4 min de leitura

OFX significa Open Financial Exchange. É uma especificação de dados para a troca eletrônica de informações financeiras entre instituições financeiras, empresas e consumidores. Desenvolvido em conjunto pela CheckFree, Intuit e Microsoft em 1997, o OFX tornou-se o formato padrão usado mundialmente para importar transações bancárias em softwares de finanças pessoais e contabilidade.

Por que o OFX é Importante

Quando você baixa seu extrato bancário, normalmente recebe um PDF — ótimo para leitura, mas inútil para softwares. O OFX é legível por máquina: contém dados de transações estruturados que aplicativos podem importar, categorizar e analisar diretamente. Sem digitação manual, sem planilhas complicadas.

Quais Aplicativos Suportam OFX?

O OFX é suportado por praticamente todos os principais aplicativos de finanças pessoais:

  • Quicken (suporte completo ao OFX desde 1997)
  • YNAB (You Need A Budget)
  • GnuCash (gratuito e open source)
  • Banktivity (macOS/iOS)
  • Moneydance
  • HomeBank (gratuito, multiplataforma)
  • Microsoft Money (legado)

Como é um Arquivo OFX?

Os arquivos OFX usam uma estrutura baseada em tags similar ao XML. Cada transação é envolvida por tags STMTTRN e inclui data, valor, tipo (DEBIT ou CREDIT) e uma descrição. O arquivo também inclui informações da conta como ID do banco, número da conta e moeda.

Como Obter um Arquivo OFX do Seu Banco

Muitos bancos permitem baixar extratos em formato OFX diretamente pelo internet banking. Porém, serviços internacionais como o Wise exportam apenas em PDF ou XML (camt.053). É aí que nosso conversor entra — ele lê seu PDF ou XML e gera um arquivo OFX válido pronto para importação.

Perguntas Frequentes

OFX é o mesmo que QFX?

QFX é uma variante proprietária do Quicken baseada no OFX. São quase idênticos na estrutura; a maioria dos apps que aceitam OFX também aceita QFX.

Posso abrir um arquivo OFX no Excel?

Não diretamente, mas você pode renomeá-lo para .txt e abrir para inspecionar o conteúdo. Para análise adequada, importe-o primeiro em um app de finanças.
